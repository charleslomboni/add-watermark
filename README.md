# add-watermark
Add custom watermark to pdfs

## Usage
```
~/Desktop/add-watermark$ python add-watermark.py -h
usage: add-watermark.py [-h] [-w WATERMARKFILE] [-d DIRECTORY]

optional arguments:
  -h, --help            show this help message and exit
  -w WATERMARKFILE, --watermarkFile WATERMARKFILE
                        Watermark.pdf to merge
  -d DIRECTORY, --directory DIRECTORY
                        Path with all pdf files
```
