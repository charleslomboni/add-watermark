# add-watermark
Add custom watermark to pdfs

## Usage
```
~/Desktop/add-watermark$ python add-watermark.py -h
usage: add-watermark.py [-h] [-w WATERMARKFILE] [-d DIRECTORY] [-s OPSYSTEM]

optional arguments:
  -h, --help            show this help message and exit
  -w WATERMARKFILE, --watermarkFile WATERMARKFILE
                        Watermark.pdf to merge
  -d DIRECTORY, --directory DIRECTORY
                        Path with all pdf files
  -s OPSYSTEM, --opSystem OPSYSTEM
                        Operation System by default is linux, if you are
                        runing on windows use .: -s win
```
